# Contributing

Thanks for your interest in this project! This is primarily a personal portfolio site, but bug reports, accessibility fixes, and small improvements are welcome.

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/<your-username>/prakhar-portfolio.git`
3. Install dependencies: `npm install`
4. Create a branch: `git checkout -b fix/short-description`
5. Run the dev server: `npm run dev`

## Making Changes

- Keep changes focused — one fix or feature per pull request.
- Match the existing code style (TypeScript, Tailwind utility classes, functional components).
- Run `npm run lint` before committing.
- Test your changes in the browser at a few viewport widths (mobile, tablet, desktop).

## Commit Messages

Use clear, descriptive commit messages, ideally following [Conventional Commits](https://www.conventionalcommits.org/):

```
fix: correct alt text on hero profile image
feat: add dark/light theme toggle
docs: update setup instructions in README
```

## Submitting a Pull Request

1. Push your branch and open a pull request against `main`.
2. Describe what changed and why.
3. Link any related issue.
4. Be responsive to review feedback — this helps things merge faster.

## Reporting Bugs

Please open an issue using the bug report template and include:

- Steps to reproduce
- Expected vs. actual behavior
- Browser/OS, and a screenshot if it's visual

## Code of Conduct

This project follows the [Code of Conduct](./CODE_OF_CONDUCT.md). Please be respectful in issues and pull requests.
